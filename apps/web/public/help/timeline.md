# Timeline

> 👑 **Owner uniquement.**
> *Pour les guests, l'équivalent est la page d'accueil avec son défilement chronologique.*

La Timeline est une **vue panoramique de tout ton journal**, contrairement à la page Journal qui ne montre qu'un seul jour à la fois. C'est l'endroit pour relire, fouiller, filtrer.

## À quoi ça sert

- Relire en continu plusieurs jours, semaines ou mois d'écriture
- Filtrer par type, tag, mood, visibilité, période
- Faire des actions en lot sur plusieurs notes (sélection multiple)
- Trouver une note précise par recherche texte

## La structure de la page

La timeline regroupe les notes **par jour**, avec un en-tête au format français (ex : « lundi 5 mai »). Les jours s'enchaînent verticalement.

Sous chaque séparateur de date, si tu as renseigné un **[daily log](journal.md)** ce jour-là, un mini badge centré le résume (`🙂 ☀️ 😴 7h ⚡ 4/5`). Clic dessus → ouvre Home à cette date pour éditer. Côté Confidant, le badge apparaît aussi (lecture seule).

### Filtres disponibles

Les mêmes filtres que sur la page Journal, mais qui s'appliquent **à toutes tes archives** :

- **Recherche texte** — instantanée, cherche dans contenu, titre, métadonnées
- **Tri** : Heure ↓/↑ (heure de la note dans la journée) ou Ajouté ↓/↑ (date de création)
- **Types de note** — un filtre par type présent dans tes archives
- **Tags** — tous les tags que tu as utilisés
- **Mood** — humeurs enregistrées
- **Visibilité** — 🔒 Privé / 🌐 Partagé / 🤝 Spécifique
- **Pour toi (confident)** — 💌
- **Secret** — 🔐
- **Section** — matin / après-midi / fin d'après-midi / soir / nuit

Les filtres se cumulent. Un compteur s'affiche à côté de chaque pour t'indiquer combien de notes correspondent.

### Filtres contextuels

Apparaissent uniquement quand pertinent :

- **Capsules** — badge avec horloge ⏰. Affiche les notes avec une date de déverrouillage future.
- **Lu par le confident** — œil 👁 avec compteur. Apparaît seulement si ton confidant a déjà lu au moins une note.

## Mode sélection multiple

Bouton avec l'icône de damier. Active les cases à cocher sur toutes les cartes. Une barre d'actions s'affiche en bas pour traiter plusieurs notes en une fois :

- Basculer en brouillon / publier
- Changer la visibilité
- Marquer/démarquer « Pour toi »
- Définir une humeur commune
- Ajouter ou retirer un tag

C'est très utile après une session d'écriture pour, par exemple, partager d'un coup toutes les notes d'un voyage avec ton confidant.

## 💡 Astuces

- **Indicateur de synchro** — quand tu vois un petit ↻ qui tourne dans l'en-tête, l'app est en train de pousser tes modifications vers le serveur.
- La **recherche est instantanée** : pas besoin de valider, le filtrage se fait à chaque frappe.
- Tu peux **cumuler tous les filtres** que tu veux (par exemple : type=Livre + tag=#sci-fi + visibilité=Privée + cette année).
- L'option **« Section »** (matin / après-midi…) est utile si tu as l'habitude de noter à des moments précis de la journée.
- Cliquer sur une note **l'ouvre en place** — pas de modal en plein écran, contrairement à la vue Guest.

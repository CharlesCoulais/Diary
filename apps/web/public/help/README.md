# Centre d'aide

Bienvenue dans la documentation de Diary. Cette aide explique chaque page, ce que tu peux y faire selon ton rôle, et les astuces pas toujours évidentes.

## Comment lire cette aide

Chaque page utilise des badges pour indiquer qui peut faire quoi :

- 👑 **Owner** — le propriétaire du journal
- 🤝 **Confidant** — un guest avec l'accès « confident »
- 👤 **Guest** — un guest avec l'accès standard
- 🌍 **Tous** — tout le monde, owner et guests

## Tu ne trouves pas ?

Si une fonctionnalité te semble manquante ou pas claire, il y a de bonnes chances qu'elle soit cachée derrière un long-press, un swipe ou un filtre conditionnel. Va jeter un œil dans la section « Astuces » de la page concernée.
